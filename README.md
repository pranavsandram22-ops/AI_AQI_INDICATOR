# 🌍 AirGuardianAI

An AI-powered air quality monitoring platform that integrates **Google Earth Engine**, **Sentinel-5P satellite data**, and **Machine Learning** to predict the Air Quality Index (AQI) and analyze atmospheric pollutants across Indian states.

## 🚀 Features

* 🛰 Live Sentinel-5P satellite data
* 🤖 AI-based AQI prediction
* 📍 State-wise air quality monitoring
* 📅 Custom date range selection
* 🌫 Pollutant analysis (CO, NO₂, SO₂, O₃, HCHO)
* 📊 Interactive Streamlit dashboard

## 🛠 Tech Stack

* Python
* Streamlit
* Google Earth Engine
* Sentinel-5P
* Scikit-learn
* Pandas
* NumPy
* Plotly
* Folium

```bash
git clone https://github.com/YOUR_USERNAME/AirGuardianAI.git
cd AirGuardianAI

pip install -r requirements.txt

streamlit run app.py
```
